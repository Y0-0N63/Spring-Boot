src/main/java 하위 패키지에 OOOApplication.java 파일이 프로젝트 생성 시 자동 생성되어 있음 (OOO = 프로젝트 이름)
해당 클래스에는 main 메서드가 존재해 실행되면, 내장된 Tomcat 서버가 배포를 시작함
(브라우저가 자동으로 열리지 않기 때문에 브라우저 주소창에 입력해 직접 접속해야 함!)
