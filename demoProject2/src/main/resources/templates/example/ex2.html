요청 위임된 request에 존재하는 파라미터 출력하기

${param.key}
- request에 존재하는 parameter 값 얻어와 출력
- 요청 URL에 쿼리스트링(GET)이나 폼 데이터(POST)로 전달된 파라미터 값을
직접 req와 같은 request scope에 세팅하지 않아도 바로 접근할 수 있도록 해줌
  • 제출된 이름
  • 제출된 나이
  • 체크된 색상

th:text / th:utext

th:text = "속성값"
- 해당 태그에 "속성값"을 내용으로 출력
- 단, html 태그와 특수문자는 해석하지 못함(innerText)

th:utext = "속성값"
- 해당 태그에 "속성값"을 내용으로 출력
- html 태그와 특수문자를 해석할 수 있음(innerHTML)

- 해당 요소의 내용으로 "문자열" + 변수값 출력
